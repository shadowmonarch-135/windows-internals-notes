# Windows Internals Notes

Welcome to my **Windows Internals Notes** repository.

This repository documents my journey while studying the Windows operating system from both a system programming and offensive security perspective. The goal is to understand how Windows works internally rather than simply memorizing APIs or attack techniques.

## Objectives

* Build a solid understanding of Windows architecture.
* Learn how user-mode applications interact with the operating system.
* Study Windows internals from a low-level perspective.
* Connect Windows concepts with offensive security techniques used during red team engagements.
* Create a well-organized knowledge base for future reference.

## Repository Philosophy

These notes are written as part of my personal learning process. Every topic is explained in my own words after studying official documentation, books, research papers, and hands-on labs. Whenever possible, I include diagrams, code snippets, and practical examples to reinforce the concepts.

## References

Some of the resources used while studying include:

* *Windows Internals* by Mark Russinovich, David Solomon, Alex Ionescu, and Pavel Yosifovich
* Microsoft Learn
* Microsoft Documentation
* Windows SDK Documentation
* Sysinternals Documentation
